## Document-reconstruction
`文档重建`的主要功能为将图片转换为tex源码，不仅仅局限于印刷体数学公式图片
该项目包括以下功能：
- 图像预处理
    - 图像拼接
    - 页面扭曲纠正
    - 文本定位
    - 角点检测、透视变换
    
- OCR
    - 识别
    - 文本纠错
    
- 转tex
    - 根据OCR结果，将排版信息依托于tex代码信息做保存
